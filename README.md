# Мини-Склад
Учебная конфигурация 1С для учёта товаров.
## Состав конфигурации
- **Справочники**: Товары, Контрагенты
- **Документы**: Поступление товаров, Реализация товаров
- **Подсистема**: Мини-Склад

## Как открыть
1. Установите платформу 1С:Предприятие (версия 8.5.1 и выше)
2. Создайте новую информационную базу без конфигурации
3. Откройте базу в Конфигураторе
4. Загрузите конфигурацию из файла (Конфигурация → Загрузить конфигурацию из файла...)
5. Выберите файл `1Cv8.cf`
